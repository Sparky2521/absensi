================================================================================
                              TROUBLESHOOTING
================================================================================

Masalah Umum dan Solusi:

1. Error: "Module not found: '@supabase/ssr'"
   Solusi: npm install @supabase/ssr

2. Error: "relation does not exist"
   Solusi: Jalankan supabase-schema.sql di Supabase SQL Editor

3. Error: "Invalid login credentials"
   Solusi: Pastikan admin user sudah dibuat via SQL

4. Error saat npm install
   Solusi: Hapus node_modules, lalu npm install lagi

5. Page tidak load
   Solusi: Restart dev server (Ctrl+C, lalu npm run dev)

Untuk troubleshooting lengkap, baca QUICKSTART.md atau SETUP.md
